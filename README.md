# VECTOR Rocketry  
### Zenith Club · IIT Ropar

VECTOR is the student rocketry team under **Zenith Club, Indian Institute of Technology Ropar**.

This repository contains the **avionics code, CAD designs, documentation, and media** related to the design and development of experimental sounding rockets by the VECTOR team.
